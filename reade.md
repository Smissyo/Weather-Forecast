# 🌤️ Weather Forecast

A full-stack weather web application engineered to deliver real-time environmental forecasts, atmospheric tracking, and rain predictions wrapped in an ultra-modern, dark-themed user interface.

---

## 💎 Project Features

* **Server-Side API Shielding:** Keeps proprietary developer keys secured on the backend environment rather than exposing them to browser inspection tools.
* **Deterministic Tomorrow-Forecast Filtering:** Aggregates and filters 3-hour interval forecast objects natively within Node to isolate a concise atmospheric portrait for tomorrow.
* **Responsive Visual Interface:** Features scalable card designs with clean layout spacing, custom SVG icons, and a dark visual theme.
* **Optimized Server-Side Rendering:** Powered by EJS to seamlessly transition the main page layout from a placeholder "Ready for Your Search" view into live weather metrics.

---

## ⚙️ Core Technical Stack

* **Front End Layers:** EJS Templates, Custom CSS Variables
* **Server Runtime:** Node.js, Express.js (ES Modules Configuration)
* **HTTP Client:** Axios
* **Configuration Protection:** Dotenv Framework
